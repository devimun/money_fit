# 업데이트 계획서 - v1.4.7

## 개요

이번 업데이트는 두 가지 주요 목표를 가진다:
1. **테마 시스템 리팩토링**: 사용자 커스터마이징 가능한 테마 시스템 구축
2. **유럽 시장 확장**: 주요 유입 국가 언어 지원

---

## Part 1: 테마 시스템 리팩토링 (완료)

### 배경

기존 테마 시스템은 하드코딩된 색상 클래스(`LightAppColors`, `DarkAppColors`)를 사용하여 런타임 커스터마이징이 불가능했다. 사용자 경험 향상을 위해 완전한 테마 커스터마이징 기능이 필요했다.

### 기술적 접근 방식

1. **ThemeExtension 기반 설계**
   - `AppThemeColors`: 22개 의미론적 색상 속성
   - `copyWith`, `lerp` 메서드로 테마 전환 애니메이션 지원

2. **ColorSeed 기반 자동 생성**
   - 하나의 seed 색상으로 전체 팔레트 자동 생성
   - Light/Dark 모드별 적절한 밝기 자동 조정
   - 대비 색상 자동 계산 (가독성 보장)

3. **Riverpod 상태 관리**
   - `themeSeedColorProvider`: 현재 테마 색상
   - `fontSizeProvider`: 폰트 크기 스케일
   - `themeRepositoryProvider`: 영구 저장소 접근

4. **BuildContext Extension**
   - `context.colors.brandPrimary` 형태로 간결한 접근
   - 보일러플레이트 코드 최소화

### 주요 성과

| 항목 | 결과 |
|------|------|
| 색상 속성 | 22개 의미론적 색상 정의 |
| 프리셋 색상 | 8개 제공 |
| 폰트 크기 옵션 | 3단계 (0.85x, 1.0x, 1.15x) |
| 테스트 커버리지 | 50+ 테스트 케이스 |
| 마이그레이션 | 모든 주요 화면 완료 |

## Part 2: 유럽 시장 확장 (언어 현지화)

최근 데이터(2026.01.01 ~ 01.15) 분석 결과, 스페인, 폴란드, 우크라이나, 체코 등 유럽 국가에서의 사용자 유입이 급증했습니다. 해당 국가 사용자들의 리텐션과 참여도를 높이기 위해 주요 유입 국가의 언어 지원(현지화)을 최우선으로 진행합니다.

## 대상 언어 및 우선순위

| 단계      | 우선순위                | 대상 국가        | 로케일 코드 |
| :-------- | :---------------------- | :--------------- | :---------- |
| **1단계** | 최우선 (Must Have)      | 스페인, 폴란드   | `es`, `pl`  |
| **2단계** | 우선 (Should Have)      | 우크라이나, 체코 | `uk`, `cs`  |
| **3단계** | 잠재적 (High Potential) | 독일, 이탈리아   | `de`, `it`  |

## 기술적 접근 방식

1.  **현지화 (Localization)**:
    - `app_ko.arb`를 기준으로 Gemini를 활용하여 각 언어별 ARB 파일 생성.
    - 특히 굴절어가 많은 슬라브어권(폴란드, 우크라이나, 체코)의 텍스트 길이 및 특수 문자 처리 주의.
2.  **UI/UX 개선**:
    - **설정 화면**: 언어 선택 옵션 추가.
    - **UI 대응**: 독일어 및 유럽 언어의 긴 텍스트로 인한 레이아웃 깨짐(Overflow) 방지.
    - **공지**: 신규 언어 추가 소식을 알리는 업데이트 안내 또는 툴팁 제공.

## 주요 모니터링 지표

- 스페인 및 폴란드 사용자의 리텐션 변화.
- 우크라이나 및 체코 사용자의 평균 참여 시간 증가율.
- 타겟 지역의 신규 사용자 대비 활성 사용자 전환율.
