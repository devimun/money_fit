# MoneyFit

MoneyFit은 사용자가 자신의 소비 습관을 쉽게 파악하고 관리할 수 있도록 돕는 가계부 앱입니다. 일일 예산을 설정하여 당일 지출이 가능한 변동 지출을 파악하게 하여, 캘린더와 지출 내역를 통해 소비 패턴을 한눈에 파악할 수 있습니다.

## 주요 기능

### 0. 온보딩

- **앱 소개**: 앱의 주요 기능과 가치를 소개하는 온보딩 스크린을 통해 사용자의 초기 사용 경험을 돕습니다.
- **초기 데이터 설정** : 앱 기능 사용에 필요한 일일 예산을 설정합니다.

### 1. 메인 화면

- **일일 지출 현황**: 오늘 사용한 금액과 남은 예산을 원형 프로그레스바로 시각화하여 보여줍니다.
- **월평균 변동 지출 및 연속 목표 성취일**: 월별 평균 지출액과 연속으로 목표를 달성한 날짜를 표시하여 소비 습관을 관리하는 데 동기를 부여합니다.
- **오늘의 지출 보기**: 오늘의 지출 내역을 간략하게 확인할 수 있는 바텀 시트 모달을 제공합니다.
- **지출 등록**: 새로운 지출을 간편하게 등록할 수 있는 바텀 시트 모달 형식의 입력 폼을 제공합니다.

### 2. 캘린더

- **월별 지출 내역**: 캘린더 형식으로 일별 지출액을 표시하여 한 달의 소비 흐름을 쉽게 파악할 수 있습니다.
- **월간 통계**: 월간 변동 지출, 필수 지출, 목표 성공/실패/연속 성공일 통계를 제공하여 성과를 추적할 수 있습니다.
- **일별 상세 내역**: 특정 날짜를 선택하면 해당일의 바텀 시트 모달 형식의 상세 지출 내역을 확인할 수 있습니다.

### 4. 설정

- **일일 예산 설정**: 사용자가 직접 일일 예산을 설정하고 변경할 수 있습니다.
- **다크 모드**: 사용자의 시력 보호와 배터리 절약을 위해 다크 모드 옵션을 제공합니다.
- **알림 설정**: 예산 초과 경고 등 주요 알림을 켜고 끌 수 있습니다.
- **데이터 관리**: 앱에 기록된 모든 데이터를 초기화할 수 있습니다.
- **앱 정보**: 버전 정보, 공지사항, 개인정보 처리방침 등 앱과 관련된 다양한 정보를 확인할 수 있습니다.
